# 🚗 Otopark Otomasyonu

## 📌 Proje Hakkında

Bu proje, **ASP.NET Web Forms**, **C#** ve **SQL Server** kullanılarak geliştirilmiş bir otopark otomasyon sistemidir.
Sistem sayesinde otoparka giriş yapan araçların bilgileri kayıt altına alınabilir ve yönetilebilir.
---
## ✨ Özellikler
- 🚗 Araç giriş kayıt işlemleri
- 👤 Araç sahibine ait bilgilerin kaydedilmesi
- 🚙 Araç tipi ve araç sınıfı bilgileri
- 💾 SQL Server veritabanı desteği
- 🖥️ ASP.NET Web Forms tabanlı kullanıcı arayüzü
---
## 🛠️ Kullanılan Teknolojiler
- ASP.NET Web Forms
- C#
- SQL Server
- HTML
- CSS
- JavaScript
---
## ⚙️ Kurulum
1. Projeyi bilgisayarınıza indirin.
2. SQL Server üzerinde gerekli veritabanını oluşturun.
3. `Web.config` dosyasındaki bağlantı (Connection String) bilgisini kendi SQL Server ayarlarınıza göre düzenleyin.
4. Visual Studio ile projeyi açın.
5. Projeyi çalıştırın.
