# 🚗 Otopark Otomasyonu

## 📌 Proje Hakkında

Bu proje, ASP.NET Web Forms ve SQL Server kullanılarak geliştirilmiş basit bir otopark otomasyon uygulamasıdır.

Uygulama üzerinden otoparka giriş yapan araçların bilgileri veritabanına kaydedilmektedir.

## ✨ Özellikler

- Araç giriş kaydı oluşturma
- TC Kimlik No, Ad ve Soyad bilgilerini kaydetme
- Araç tipi bilgisi
- Araç sınıfı bilgisi
- SQL Server veritabanına kayıt

## 🛠️ Kullanılan Teknolojiler

- ASP.NET Web Forms
- C#
- SQL Server
- HTML
- CSS
